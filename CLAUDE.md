# Claude Configuration

Always use Bun as the package manager.
Always run `bun install` to install dependencies.
Always use `turbo run dev` to start development servers.
Always use `turbo run build` to build the project.
Always use `turbo run lint` to lint the project.
Always use `turbo run format` to format the project.
Always use `turbo run clean` to clean the project.
